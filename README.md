# Gym APP API

Service created for application (client) that can be used by gyms.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install
```

## Dependencies

```json
"dependencies": {
    "body-parser": "^1.19.0",
    "consign": "^0.1.6",
    "mongoose": "^5.12.2",
    "mongodb": "^3.6.5"
}
```
